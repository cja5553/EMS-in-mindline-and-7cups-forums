# EMS-in-mindline-and-7cups-forums

Codes for manuscript titled "AI-aided Population-Level Comparisons of EarlyMaladaptive Schemas Between Singaporean and Global Online Communities". 

Note:
- The mindline and 7cups data used this study is available at https://osf.io/6p3qg/overview
- EMS prediction and evaluation code is available at https://github.com/NUS-IDS/ems_mentalhealth/tree/npj26
- The benchmark dataset used for selecting LLMs is described in https://aclanthology.org/2023.findings-emnlp.792.pdf
